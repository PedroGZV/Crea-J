Adaptador Apertura de cajón USB BT-100U

Especificaciones técnicas:

  • Adaptador apertura de cajón USB BT-100U
  • De RJ11 a USB.
  • Incluido cable de conexión USB.
  • El adapatador de cajón USB BT-100U esta diseñado para proporcionar un conector usb para los cajones estandar RJ-11.
  • Esto permite conectar el cajón directamente a un puerto USB del ordenador.
  • Esto significa que podemos conectar el cajón al ordenador sin necesidad de usar una impresora.
  • El adaptador BT-100U utiliza un driver de puerto serie virtual.
  • El sotfware reconoce el cajon como un puerto serie estandar RS-232.
  • De esta forma el sotfware puede comunicarse con el cajón como si estuviera conectado a un puerto serie.

    • Caracteristicas del producto:

    • No requiere alimentacion externa. USB autoalimentado
    • No es necesario configurar las propiedades del puerto COM.
    • Funciona con cualquier comando enviado al cajón. Escribe cualquier dato al puerto COM del cajón, y el cajón se abrirá.
    • Dimensiones: 68x44x25mm
    • Peso: 80 gr (Incluye Cable)
    • Color:Negro
    • Conector del PC: USB
    • Conector Cajón: RJ11 (12V, 24V)
