BT-100U USB Drawer Opening Adapter

TECHNICAL FEATURES

  • BT-100U USB drawer opening adapter.
  • from RJ11 to USB.
  • USB connection cable included.
  • The BT-100U USB Drawer Adapter is designed to provide a USB connector for standard RJ-11 drawers.
  • This allows you to connect the drawer directly to a USB port on your computer.
  • This means that we can connect the drawer to the computer without using a printer.
  • The BT-100U adapter uses a virtual serial port driver.
  • The software recognizes the drawer as a standard RS-232 serial port.
  • In this way the software can communicate with the drawer as if it were connected to a serial port.

    • Product features:

    • It does not require external power. Self powered USB
    • You do not need to configure the COM port properties.
    • It works with any command sent to the drawer. Write any data to the drawer's COM port, and the drawer will open.
    • Dimensions: 68x44x25mm
    • Weight: 80 gr (Includes Cable)
    • Color: black
    • PC connector: USB
    • Drawer connector: RJ11 (12V, 24V)
